# Excel 快捷键 Top200

## 文件操作

| 功能         | Windows 快捷键 | Mac 快捷键   |
|--------------|----------------|--------------|
| 新建         | `Ctrl + N`     | `Cmd + N`    |
| 打开         | `Ctrl + O`     | `Cmd + O`    |
| 保存         | `Ctrl + S`     | `Cmd + S`    |
| 另存为       | `Ctrl + Shift + S` | `Cmd + Shift + S` |
| 打印         | `Ctrl + P`     | `Cmd + P`    |
| 关闭文件     | `Ctrl + W`     | `Cmd + W`    |
| 退出         | `Alt + F4`     | `Cmd + Q`    |
